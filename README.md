## ✨ Features

- **📊 OEE Calculator:** Instant calculation of Availability, Performance, Quality, and Overall OEE
- **🏭 Machine Management:** Add, edit, and track multiple production machines
- **📝 Shift Logging:** Record production shifts with detailed metrics
- **📈 Interactive Charts:** OEE trends, machine comparisons, loss distribution
- **⚠️ Six Big Losses Analysis:** Breakdown, Setup, Idling, Speed, Defects, Yield
- **🎯 Target Setting:** Configurable OEE targets with visual benchmarks
- **💾 Backup & Restore:** One-click JSON backup and restore
- **📥 Multi-Format Export:** Excel, PNG, PDF (Pro version)
- **🔒 100% Offline:** All data stored locally in IndexedDB—never leaves your machine
